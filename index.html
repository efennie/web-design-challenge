    Max Temperature Humidity Cloudiness Windspeed
The purpose of this project was to use API weather data to prove that temperature increases as you get closer to the equator. We all learned this growing up but using API weather data we can show the correlation between temperature and latitude. To gather weather data we used OpenWeatherAPI which allowed us to randomly pull weather data from different cities at different latitudes to run our analysis. After some data cleansing and wrangling, I created a dataframe in python to be able to graph my findings. The visualizations are the result of using matplotlib to plot the latitude versus different weather variables. After plotting latitude versus cloudiness, windspeed, humidity, and max temperature we are able to confirm our initial hypothesis that temperture does increase towards the equator.
